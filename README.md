# Quiz-Game

A simple beginner-friendly Python quiz game that asks the user basic computer-related questions and keeps track of their score.

## Project Overview

This project is a text based quiz program built using Python. When the program starts, it welcomes the user and asks if they want to play. If the user types "yes", the quiz begins and asks 6 questions about common computer terms:

 - CPU
 - GPU
 - RAM
 - PSU
 - SSD
 - OS

At the end, the program shows:
 - How many questions were answered correctly.
 - The final percentage score.

## How It Works

 1. The program prints a welcome message.
 2. It asks the use if they want to play.
 3. If the user does not type 'yes', the program ends.
 4. If the user types 'yes', the quiz starts.
 5. The user answers 6 computer-related questions.
 6. Each correct answer adds 1 point to the score.
 7. After all questions are answered, the final score and percentage are displayed.

## How to Run the Project

1. Make sure Python is installed on your computer.
2. Copy the code into a file named quiz.py
3. Open a terminal or command prompt.
4. Run the program with:
   python quiz.py
